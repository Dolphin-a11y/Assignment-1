# Drift

Drift is a responsive stress-relief web app that turns a quick 1–10 check-in into a personalised mindful diversion.

## Features

- Adaptive colour themes and Tone.js soundscapes
- A focus sprint for low stress, hidden-object meadow for moderate stress, and image jigsaw for high stress
- Device-local check-in history and progress chart
- Guided breathing, meditation, and ambient video options
- Keyboard, touch, responsive, and reduced-motion support

## Run locally

Requires Node.js 22.13 or newer.

```bash
pnpm install
pnpm dev
```

Create a production build with `pnpm build`.

## Privacy

Stress check-ins stay in the browser's local storage and are not sent to a server. Drift is a wellbeing aid, not a substitute for professional mental-health support.
